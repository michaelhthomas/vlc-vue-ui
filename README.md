# vlc-vue-ui

*Early Alpha*

Modern Web UI for VLC Media Player built with Vue.js and Vuetify.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
